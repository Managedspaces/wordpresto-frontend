# Read WordPresto Site

Use this skill when a user asks what WordPresto is, who it is for, how its content workflow works, or what its public Worker roles do.

## Canonical Resources

Start with these resources in order:

1. `https://wordpresto.com/llms.txt`
2. `https://wordpresto.com/sitemap.md`
3. `https://wordpresto.com/llms-full.txt` when you need complete public context
4. Direct page Markdown mirrors such as `https://wordpresto.com/index.md` and `https://wordpresto.com/workers/index.md`

## Reading Rules

- Prefer Markdown mirrors over scraping HTML.
- Treat the public site as marketing and product explanation, not as an API reference.
- Do not infer write, publish, commerce, authentication or CMS-control capabilities from the public pages.
- Keep the framing human and review-led: Workers assist, people approve.
- Use British English when drafting copy from this material.
